# ai_interviewer_ustc
## 产品简介

ai_interview_ustc是一款基于大语言模型（LLM）与语音交互技术的智能化模拟面试系统。系统旨在为求职者、学生提供沉浸式的面试训练体验，通过语音对话、知识库检索、智能评估等功能，帮助用户提升面试技巧与专业能力。

系统采用模块化架构，整合了语音识别（ASR）、语音合成（TTS）、大语言模型对话（LLM）与检索增强生成（RAG）等核心技术，支持多角色面试官设定、实时语音交互、知识库辅助问答等功能，实现了从“听”到“答”再到“评”的完整面试闭环。

---

## 核心功能
### 1. **智能语音对话**
- 支持语音输入与文字输入两种方式
- 实时语音识别（基于 StepFun ASR）
- 智能语音合成播报（基于 StepFun TTS）
- 流式对话响应，模拟真实面试节奏

### 2. **多角色面试官**
- 预设多种面试官角色：技术、算法、系统设计、行为面试、计算机基础
- 支持自定义提示词，灵活调整面试风格与难度
- 基于历史对话的上下文记忆，实现连贯追问

### 3. **知识库检索增强（RAG）**
- 支持基于领域（如计算机科学“cs”）的向量检索
- 可根据难度、主题等元数据过滤检索结果
- 可视化展示检索内容与来源

### 4. **面试过程管理**
- 完整的对话历史记录与展示
- 支持一键开始新对话
- 音频文件临时存储与自动清理
- 面试过程中的语音片段可重复播放

### 5. **友好的交互界面（Streamlit）**
- 响应式Web界面，支持移动端与桌面端
- 侧边栏集中控制面试设置
- 分页展示对话与知识检索内容

## 技术亮点
### 1. **模块化架构设计**
- audio_processor.py # 语音处理模块（ASR/TTS）
- llm_agent.py # 大语言模型对话模块
- rag_engine.py # 检索增强生成模块
- app_streamlit.py # 前端交互界面
- config.py # 统一配置管理

各模块职责清晰，耦合度低，便于扩展与维护。

### 2. **流式对话与语音合成联动**
- LLM 流式输出实时显示
- 支持整段语音合成播报
- 音频文件自动管理，避免资源泄露

### 3. **智能断句与语音触发**
- `chunking_tool` 实现基于标点的智能断句
- 为未来实时流式 TTS 预留接口
- 支持中英文混合文本处理

### 4. **异步语音处理**
- 录音、保存、转写全流程异步化
- 避免界面卡顿，提升用户体验
- 支持高并发场景下的音频处理

### 5. **可扩展的知识库系统**
- 支持多领域向量库（如 cs、business、law 等）
- 基于 Chroma + DashScope Embeddings 的检索方案
- 提供建库脚本（`build_cs_vector_store.py`）与测试工具（`test_rag.py`）

### 6. **统一的配置管理**
- API 密钥、路径、模型参数集中管理
- 支持环境变量与硬编码双模式
- 自动目录初始化，降低部署门槛
